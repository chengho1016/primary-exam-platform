# 四年級數學科考試三內容包

試卷編號：`2324-03-MA-P4`

這個資料夾是網站第一份可入庫內容，已把原本24頁PDF拆成正式列印來源、46題結構化題庫資料及獨立題目圖片。學生作答及批改頁不在列印檔內。

## 內容

- `paper.json`：試卷資料、分部、抽題規則及私隱設定。
- `questions.json`：46題題幹、答案、解析、課題、難度、圖片及網上練習設定。
- `practice-blueprint.json`：每次抽15題時的課題及難度比例。
- `print/2324-03-MA-P4-print.pdf`：只含原檔第1至12頁的空白試卷。
- `assets/questions/q01.png`至`q46.png`：從空白試卷裁出的題目圖片。
- `assets/questions/stimulus-q38-q42.png`：第38至42題共用的棒形圖。
- `review-notes.md`：內容擁有人需要確認的項目。
- `tools/crop_questions.py`：可重複產生題目圖片的裁切工具。

## 題目分佈

- 分數：20題
- 四則運算及應用題：7題
- 周界與面積：14題
- 數據處理：5題

難度標籤為容易18題、中等18題、困難10題。現時44題可進入網上抽題池；第24題只適合按原尺寸列印，第37題等待答案確認。

## 入庫原則

網上練習應讀取`questions.json`，不要直接把整頁試卷當成一題。列印功能則使用12頁空白PDF，並在伺服器端另加會員編號、日期及授權編號水印。
